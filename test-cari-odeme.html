Cari Hesap Ödemesi Test

Test Instructions
  1. Open the main application at Main Application
  2. Select "Cari Hesap Ödemeleri" from the instruction type menu
  3. Choose either "Satıcı Ödemesi" or "Müşteri Ödemesi"
  4. Verify that the recipient company list is filtered correctly:
    • For "Satıcı Ödemesi", only companies with type "Satıcı" should appear
    • For "Müşteri Ödemesi", only companies with type "Müşteri" should appear
  5. Fill in the sender information (must be a "Grup" company)
  6. Fill in the recipient information
  7. Click "Yeni Ödeme Ekle" to test multi-payment functionality
  8. Verify that new payment fields are added dynamically
  9. Fill in the payment details
  10. Click "Talimat Oluştur" to create the instruction
  11. Verify that the instruction is created and displayed correctly
Expected Results
  • Company type filtering should work correctly in the recipient dropdown
  • Dynamic payment fields should be added when clicking "Yeni Ödeme Ekle"
  • Each payment field should have its own recipient company and bank account selection
  • Each payment field should have its own amount and description fields
  • When creating the instruction, all payments should be processed and saved
  • The output should display all payments in a clear, organized format
